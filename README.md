# dotfiles
After cloning to home dir, set up links like so  
`ln -svfn ~/dotfiles/<FILE_NAME> ~`  
for the following files or directories  
* liquidprompt
* .bash_profile
* .bashrc
* .gitconfig
* .tmux.conf
* .vimrc
* .vim
* z.sh
* kubectl-aliases/.kubectl_aliases
* liquidpromptrc (link to ~/.config/)
